    기업 A의 생산함수가 Q = √(2K + L) 이다. 이에 관한 설명으로 옳은 것은? (단, Q는 산출량, K는 자본, L은 노동이다.)

    1. 1. 생산함수는 규모에 대한 수확불변이다.
    2. 2. 등량곡선의 기울기는 −4이다.
    3. 3. 두 생산요소는 완전보완재이다.
    4. 4. 등량곡선과 등비용곡선의 기울기가 다르면 비용최소화점에서 한 생산요소만 사용한다.
    5. 5. 한계기술대체율은 체감한다.

    정답 4번

    🔍 보기별 해설
    1. K와 L을 모두 t배 하면 산출량은 √(2tK+tL)=√t·√(2K+L)=√t·Q가 되어 t배가 아니라 √t배만 늘어난다. 규모를 늘릴 때 산출이 덜 늘어나는 수확체감이므로 수확불변이라는 설명은 옳지않다.
    2. 등량곡선은 2K+L을 일정하게 유지하는 선이다. 가로축을 L, 세로축을 K로 두면 K=(상수−L)/2이 되어 기울기가 −1/2이다. 따라서 −4라는 설명은 옳지않다.
    3. 이 생산함수는 2K+L처럼 두 요소가 정해진 비율로 서로 대체되는 형태라 완전대체 성격에 가깝다. 반드시 함께 투입해야 하는 완전보완재가 아니므로 옳지않다.
    4. 등량곡선이 직선(한계기술대체율 일정)이라 요소가격 비율과 기울기가 다르면 값싼 요소 한쪽만 쓰는 것이 최소비용이 된다. 모서리(구석)에서 한 요소만 사용한다는 설명은 옳다.
    5. 직선 등량곡선에서는 한 요소를 다른 요소로 바꾸는 교환비율이 어디서나 일정하다. 한계기술대체율이 일정하므로 체감한다는 설명은 옳지않다.
    🧮 풀이
    1. 기업 A의 관리자 甲이 노동과 자본을 함께 두 배로 늘려도 산출은 √2배(약 1.41배)에 그친다. 규모를 키울수록 효율이 떨어지므로 이는 수확불변이 아니라 수확체감이다.
    2. 관리자 甲이 등량곡선을 가로축 L, 세로축 K로 그리면 2K+L이 일정하도록 K=(상수−L)/2이 되어 기울기가 −1/2이 된다. 따라서 기울기가 −4라는 것은 맞지 않는다.
    3. 완전보완재라면 두 요소를 정해진 짝으로만 써야 하지만, 이 생산함수는 관리자 甲이 자본과 노동을 2:1 비율로 맞바꿔도 산출이 유지된다. 서로 대체가 가능하므로 완전보완재가 아니다.
    4. 기업 A가 노동으로 산출 1을 늘리는 비용과 자본으로 산출 1을 늘리는 비용을 비교했을 때 자본이 더 싸다면, 관리자 甲은 노동을 전혀 쓰지 않고 자본만으로 목표 산출을 채운다. 등량곡선이 직선이라 두 요소를 섞는 내부점보다 싼 요소 하나에 몰아넣는 구석해가 최소비용이 되기 때문이다.
    5. 관리자 甲이 노동을 자본으로 바꾸는 교환비율은 어느 조합에서나 1/2로 일정하다. 한계기술대체율이 변하지 않으므로 체감한다고 볼 수 없다.

    이윤을 극대화하는 독점기업 A의 평균총비용함수는 ATC = 20/Q + Q 이고, 시장수요함수는 P = 200 − 4Q 일 때, 독점이윤은? (단, Q는 거래량, P는 가격이다.)

    1. 1. 800
    2. 2. 1,600
    3. 3. 1,980
    4. 4. 2,490
    5. 5. 2,540

    정답 3번

    🔍 보기별 해설
    1. ① 총비용은 TC = ATC × Q = (20/Q + Q)×Q = 20 + Q²이고, 여기서 한계비용 MC = 2Q이다. ② 총수입은 TR = P×Q = (200 − 4Q)Q = 200Q − 4Q²이고, 한계수입 MR = 200 − 8Q이다. ③ 이윤극대화 조건 MR = MC에서 200 − 8Q = 2Q, 즉 10Q = 200이므로 Q = 20이다. ④ 가격은 P = 200 − 4×20 = 120이다. ⑤ 총수입 TR = 120×20 = 2,400, 총비용 TC = 20 + 20² = 420이므로 독점이윤 = 2,400 − 420 = 1,980이다. 따라서 독점이윤은 1,980이다.

    실질임금의 경기순환성에 관한 설명으로 옳은 것은?

    3. 3. 효율성임금이론은 실질임금의 경기순행성을 설명한다.
    4. 4. 실물경기변동이론에 따르면 양(+)의 기술충격은 실질임금을 상승시킨다.

    정답 4번

    🔍 보기별 해설
    1. 명목임금이 경직적일 때 총수요가 늘면(호경기) 물가가 올라 실질임금이 오히려 낮아진다. 즉 실질임금이 경기와 반대로 움직이는 경기역행적이므로, 경기순행적이라는 설명은 옳지않다.
