    🔍 보기별 해설

    2. ㄴ. 운동 중 혈당을 유지하려고 이자에서 글루카곤 분비가 늘어 간의 글리코겐 분해·포도당 방출을 촉진한다. 옳다.
    3. ㄷ. 운동 시 교감신경이 활성화되어 에피네프린은 오히려 크게 증가한다. '감소'는 반대다. 옳지 않다.

    3. 전력 질주 직전 심장이 두근거리는 것은 에피네프린이 늘어난 결과이지 줄어든 게 아니다.

    3. 더위 훈련을 거치면 혈액의 물 성분(혈장)이 늘어 순환이 넉넉해진다.

    정답 1번

    🔍 보기별 해설
    1. 산화질소는 혈관을 넓히는(혈관확장) 물질이라 혈압을 낮추는 쪽으로 작용한다. 평균동맥혈압을 올리는 요인이 아니다. 옳지 않다.
    2. 엔도테린-1은 강력한 혈관수축 물질로 말초저항과 혈압을 올린다. 평균동맥혈압 증가 요인이다. 옳다.
    3. 후부하가 커지면 심장이 밀어내야 하는 압력이 높아져 평균동맥혈압이 오른다. 옳다.
    4. 노르에피네프린은 교감신경 자극으로 혈관을 수축시켜 혈압을 올린다. 옳다.
    🧮 풀이
    1. 혈관내피에서 나오는 산화질소는 혈관을 이완시켜 혈압을 낮추는 쪽이라, 혈압을 올리는 원인으로 볼 수 없다.
    2. 엔도테린-1이 늘면 혈관이 조여 혈압이 올라간다.
    3. 甲이 건강검진에서 동맥경화가 있다는 말을 듣고, 심장이 굳은 동맥을 상대로 피를 밀어낼 때 맞서는 압력이 커지면 평균동맥압도 따라서 높아진다는 설명을 들었다.

    〈보기〉에서 유산소 트레이닝 후 생리적 적응 현상으로 옳은 것만을 모두 고른 것은? ㄱ. 최대하 운동 시 포스포프룩토키나아제(PFK)의 활성 감소 ㄴ. 최대하 운동 시 동일 시점에서의 근 글리코겐 이용률 감소 ㄷ. 최대하 운동 후 초과산소섭취량(EPOC) 증가 ㄹ. 최대 운동 시 젖산 역치 증가

    2. 2. ㄴ, ㄹ
    3. 3. ㄱ, ㄴ, ㄷ
    4. 4. ㄱ, ㄴ, ㄹ

    정답 4번

    🔍 보기별 해설

    2. ㄴ. 훈련 후 같은 강도서 지방을 더 쓰므로 근 글리코겐을 아껴, 동일 시점 글리코겐 이용률이 감소한다(글리코겐 절약 효과). 옳다.

    4. ㄹ. 훈련 후 젖산이 쌓이기 시작하는 강도(젖산 역치)가 더 높은 강도로 이동해 젖산 역치가 증가한다. 옳다.
    🧮 풀이
    1. 훈련이 잘 된 사람은 같은 페이스 달리기에서 당 대신 지방을 더 써서 해당 효소를 덜 돌린다.
    2. 훈련된 사람은 같은 강도서 근육에 저장한 글리코겐을 아껴 오래 버틴다.
    3. 훈련이 되면 운동 후 숨이 빨리 안정되어 회복에 드는 여분의 산소(EPOC)가 줄어든다.
    4. 훈련 후에는 더 빠른 속도까지 뛰어도 젖산이 크게 쌓이지 않는다.

    2. 2. ㄱ, ㄹ
    3. 3. ㄴ, ㄷ

    정답 4번

    🔍 보기별 해설
    1. ㄱ. 정맥수축이 감소하면 정맥 환류가 줄어 오히려 심박출·뇌 혈류에 불리하다. 뇌 혈류를 회복시키는 기전이 아니다. 옳지 않다.
    2. ㄴ. 뇌 혈류 자동조절은 관류압이 떨어지면 뇌혈관을 확장해 혈류를 일정하게 유지·회복시킨다. 옳다.
    3. ㄷ. 이산화탄소는 뇌혈관을 넓히는 물질이라, 이산화탄소가 '증가'해야 혈류가 는다. '감소로 증가'는 반대다. 옳지 않다.
    4. ㄹ. 뇌 조직에 H⁺·K⁺·ADP 같은 대사산물이 쌓이면 국소 혈관이 확장되어 뇌 혈류가 증가한다. 옳다.
    🧮 풀이
    1. 정맥이 덜 조여지면 심장으로 돌아오는 피가 줄어 오히려 뇌로 가는 혈류 유지에 도움이 안 된다.
    2. 혈압이 잠깐 떨어져도 뇌혈관이 스스로 넓어져 뇌 혈류를 일정하게 지킨다.
    3. 숨을 몰아쉬어 이산화탄소가 빠지면 뇌혈관이 좁아져 어지러운데, 반대로 이산화탄소가 늘면 혈관이 넓어진다.
    4. 일하는 조직에 대사산물이 쌓이면 그 부위 혈관이 벌어져 피가 더 몰린다.

    🧮 풀이
    1. 평소 운동을 안 한 사람은 뛰기 시작할 때 숨이 늦게 트여 초반 산소 부족(Ⓐ)이 더 크게 나타난다.
    2. 항정상태 1.8 L/min을 60 kg으로 나누면 30 mL/kg/min이라, 25라는 값은 맞지 않는다.
    3. 운동을 끝냈는데도 한동안 숨이 가쁜 것은 오른 체온과 에피네프린을 되돌리느라 산소를 더 쓰기 때문이다.
    4. Ⓓ 영역을 에너지로 바꿔 계산하면 보기에 적힌 31.5 kcal와 값이 어긋난다.

    1. 멜라토닌은 밤에 분비되어 잠과 생체리듬을 조절하는 호르몬으로, 스트레스 대응이나 혈당 유지와는 거리가 멀다. 문항이 찾는 호르몬이 아니므로 옳지 않다.
    2. EPO는 적혈구 생성을 늘리는 호르몬이다. 스트레스 반응이나 혈당 유지와는 관계가 없어 이 문제가 찾는 호르몬이 아니므로 옳지 않다.
    3. 코르티솔은 스트레스 반응 호르몬으로 당신생을 촉진해 혈당을 유지한다. 옳다.
    4. 항이뇨호르몬(ADH)은 수분 재흡수를 조절하며 혈당 유지와는 직접 관련이 없다. 옳지 않다.
    🧮 풀이
    1. 멜라토닌은 밤에 분비되어 잠을 유도한다.
    2. 甲이 고지대에 머무는 동안 신장에서 EPO 분비가 늘어 골수의 적혈구 생성이 촉진되면서 혈액 속 적혈구 수가 점점 많아진다.
    3. 시험·경기 같은 스트레스 상황에서 코르티솔이 혈당을 끌어올려 대응 에너지를 마련한다.
    4. ADH는 콩팥에서 물을 재흡수해 소변을 줄이는 호르몬이라 혈당과는 직접 관련이 없다.
