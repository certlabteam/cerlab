    2. 2. 급성 구획증후군(acute compartment syndrome)
    3. 3. 만성 발목 불안정(chronic ankle instability)
    4. 4. 발목굴 증후군(tarsal tunnel syndrome)

    정답 1번

    🔍 보기별 해설
    1. 아킬레스 힘줄병증은 반복적 인장응력·과부하로 힘줄이 변성(섬유화·흉터)되고, 종아리근 유연성 감소로 증상이 악화된다. 설명과 일치한다. 옳다.
    2. 급성 구획증후군은 부종·출혈로 구획 내 압력이 급격히 오르는 응급 상태로, 설명과 다르다. 옳지 않다.
    3. 만성 발목 불안정은 반복 염좌 후 발목이 자주 접질리는 상태로, 힘줄 섬유화·인장 과부하 설명과 다르다. 옳지 않다.
    4. 발목굴 증후군은 발목 안쪽에서 경골신경이 눌려 저림·통증이 생기는 것으로, 설명과 다르다. 옳지 않다.
    🧮 풀이
    1. 달리기·점프를 반복하는 러너가 아킬레스건에 미세손상이 쌓여 뻐근한 통증을 겪는 것이 아킬레스 힘줄병증이다.
    2. 급성 구획증후군은 외상 후 종아리가 급격히 붓고 극심한 통증이 오는 응급 상황이다.
    3. 만성 발목 불안정은 발목을 자주 삐끗하는 상태다.
    4. 발목굴 증후군은 발바닥·발가락이 저리는 신경 압박 증상이다.

    3. ㄷ. 앞당김 검사는 발을 앞으로 당겨 전거비인대(발목 바깥 인대) 손상을 평가한다. 옳다.

    2. 윈들라스 검사는 엄지를 위로 젖혀 발바닥 통증(족저근막염)을 확인한다.
    3. 甲이 발목을 잡힌 채 발꿈치가 앞으로 끌려 나오는 정도를 재어, 검사자가 발목 바깥 앞쪽 인대인 전거비인대 상태를 살핀다.
    4. 발을 안쪽으로 기울여 벌어지는 각도를 보는 talar tilt 검사로 외측 인대 손상을 확인한다.

    〈보기〉에서 제시된 조직부하(tissue loading) 유형이 동일한 것만을 모두 고른 것은? ㄱ. 발등굽힘에 의한 아킬레스힘줄 손상 발생 ㄴ. 피부에서 포진, 찰과상 발생 ㄷ. 넙다리곧은근의 과신장(overstretching) 손상 발생 ㄹ. 반복적인 점프 동작에 의한 척추 디스크 손상 발생

    1. 1. ㄱ, ㄷ
    2. 2. ㄱ, ㄹ
    3. 3. ㄴ, ㄷ
    4. 4. ㄴ, ㄹ

    정답 1번

    🔍 보기별 해설
    1. ㄱ. 발등굽힘으로 아킬레스힘줄이 늘어나며 손상되는 것은 조직을 잡아당기는 '인장(tension)' 부하다. 정답 조합에 해당한다. 옳다.
    2. ㄴ. 피부의 포진·찰과상은 표면이 스치며 생기는 '전단·마찰(shear)' 부하로, 인장과 다른 유형이다. 옳지 않다.
    3. ㄷ. 넙다리곧은근의 과신장 손상도 근육을 잡아당기는 '인장(tension)' 부하다. ㄱ과 같은 유형으로 정답에 해당한다. 옳다.
    4. ㄹ. 반복 점프로 인한 척추 디스크 손상은 위에서 눌리는 '압박(compression)' 부하로, 인장과 다른 유형이다. 옳지 않다.
    🧮 풀이
    1. 甲이 발끝을 정강이 쪽으로 힘껏 당기자 팽팽해진 아킬레스힘줄이 늘어나며 다치는데, 이는 세로로 잡아끄는 인장 손상이다.
    2. 피부가 쓸려 생기는 찰과상은 표면을 문지르는 힘(전단·마찰) 때문이다.
    3. 甲이 전력 질주 도중 허벅지 앞 넙다리곧은근이 한계를 넘어 늘어나며 찢어진 것도 근육을 잡아끄는 인장 부하 손상이다.
    4. 점프 착지에서 척추가 위아래로 눌려 디스크가 다치는 것은 압박이다.

    〈보기〉에서 신경학적 검사 중 깊은 힘줄 반사(deep tendon reflex)의 평가에 대한 설명으로 옳은 것만을 모두 고른 것은? ㄱ. 평가 결과는 1-5까지 다섯 등급이다. ㄴ. 신경 뿌리 수준(nerve root level) L2는 평가할 수 없다. ㄷ. 정상등급은 2등급이다. ㄹ. 신경 뿌리 수준 C5는 근육피부신경(musculocutaneous)을 평가한다.

    1. 1. ㄱ, ㄴ
    2. 2. ㄴ, ㄷ
    3. 3. ㄴ, ㄹ
    4. 4. ㄷ, ㄹ

    정답 4번

    🔍 보기별 해설
    1. ㄱ. 깊은 힘줄 반사 등급은 0∼4의 다섯 단계(0,1,2,3,4)로 매긴다. '1-5'는 틀리다. 옳지 않다.
    2. ㄴ. L2 신경 뿌리는 무릎 관련 반사 등으로 평가할 수 있다. '평가할 수 없다'는 틀리다. 옳지 않다.
    3. ㄷ. 깊은 힘줄 반사 등급은 0에서 4까지 매기며, 그 가운데 반응이 정상적으로 나오는 중간 단계가 2등급이다. 옳다.

    2. L2 수준도 무릎 주변 반사로 평가할 수 있어 '평가 불가'는 틀린 말이다.
    3. 정상적으로 반사가 적절히 나오는 상태를 2등급으로 본다.
    4. 甲이 팔꿈치 앞쪽 위팔두갈래근 힘줄을 두드리자 팔이 굽는 반사가 나타나, 이는 근육피부신경이 지나는 C5 수준을 확인해 준다.

    신경근 수준(nerve root level)에 대한 운동검사(motor testing)로 옳은 것은?

    1. 1. C2 – 팔꿈치 관절 굽힘과 손목 폄(elbow flexion and wrist extension)
    2. 2. C3 – 어깨 벌림(shoulder abduction)
    3. 3. C4 – 어깨 올림(shoulder shrug)
    4. 4. C5 – 목 가쪽굽힘(neck lateral flexion)

    정답 3번

    🔍 보기별 해설
    1. 팔꿈치 굽힘은 C5∼C6, 손목 폄은 C6 수준이다. C2와 짝지은 것은 옳지 않다. 옳지 않다.
    2. 어깨 벌림은 C5 수준이다. C3와 짝지은 것은 옳지 않다. 옳지 않다.
    3. 어깨 올림(shoulder shrug, 승모근)은 C3∼C4 수준으로 검사한다. C4와 짝이 맞다. 옳다.
    4. 목 가쪽굽힘은 상위 경추(C3 등) 수준이다. C5와 짝지은 것은 옳지 않다. 옳지 않다.
    🧮 풀이
    1. 甲이 아령을 들며 팔꿈치를 굽히는 힘은 C5와 C6가, 손목을 뒤로 젖히는 폄은 C6가 맡아 위쪽 목신경과는 짝이 되지 않는다.
    2. 甲이 두 팔을 옆으로 들어 어깨 높이까지 벌리는 동작은 C5가 담당하는 기능이라 더 위쪽 목신경과 짝지을 수 없다.
    3. 어깨를 으쓱 올리는 동작으로 C3~C4 수준을 검사한다.
    4. 목을 옆으로 기울이는 것은 상위 경추 기능이라 C5가 아니다.
