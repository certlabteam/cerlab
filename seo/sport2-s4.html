    성인의 1 MET가 의미하는 것은?

    1. 1. V̇CO₂
    2. 2. V̇O₂ rest
    3. 3. V̇O₂ max
    4. 4. V̇O₂ peak

    정답 2번

    🔍 보기별 해설
    1. V̇CO₂는 이산화탄소 배출량이다. 1 MET가 의미하는 안정 시 산소섭취량과는 다르다. 옳지 않다.
    2. 1 MET는 안정 시 산소섭취량(V̇O₂ rest, 약 3.5 mL/kg/min)을 의미한다. 옳다.
    3. V̇O₂ max는 최대 산소섭취량이다. 안정 시 값을 뜻하는 1 MET와는 다르다. 옳지 않다.
    4. V̇O₂ peak는 검사에서 도달한 최고 산소섭취량이다. 1 MET와는 다르다. 옳지 않다.
    🧮 풀이
    1. 甲이 운동 중 내쉰 숨에서 이산화탄소가 얼마나 나오는지 재는 값이 V̇CO₂임을 알게 되었는데, 이는 몸이 뿜어내는 이산화탄소의 양일 뿐 가만히 쉴 때의 산소 소비량을 뜻하는 1 MET와는 전혀 다른 지표였다.
    2. 甲은 소파에 가만히 앉아 쉬는 동안에도 몸이 기본적으로 산소를 소비한다는 것을 배웠는데, 이때 체중 1kg당 1분에 약 3.5mL의 산소를 쓰는 안정 시 산소섭취량이 바로 1 MET의 기준이 된다는 설명을 들었다.
    3. 甲이 트레드밀에서 숨이 턱에 차도록 전력을 다해 달릴 때 몸이 최대로 빨아들이는 산소량이 V̇O₂ max인데, 이는 한계까지 몰아붙였을 때의 값이라 편히 쉴 때의 기준인 1 MET와는 크게 벌어졌다.
    4. 甲이 점증 부하 검사를 받으며 더는 못 버틸 지점까지 갔을 때 기록된 가장 높은 산소섭취량이 V̇O₂ peak였는데, 이는 검사 중 실제로 도달한 최고치일 뿐 안정 시 소비량인 1 MET와는 구별되었다.

    〈보기〉에서 설명하는 것은? 쾌적한 환경(온도 21℃, 습도 50%)에서 일정한 강도(50% V̇O₂ max)로 60분 동안 운동 시 35~45분 후에 체온은 고원(plateau)에 도달하여 일정하게 유지된다.

    1. 1. 항정상태(steady state)
    2. 2. 항상성(homeostasis)
    3. 3. 정적 피드백(positive feedback)
    4. 4. 부적 피드백(negative feedback)

    정답 1번

    🔍 보기별 해설
    1. 일정 강도 운동에서 체온이 고원에 도달해 일정하게 유지되는 것은 항정상태다. 옳다.
    2. 항상성은 몸 전체의 내부 환경을 일정하게 지키려는 넓은 개념이다. 특정 값이 운동 중 일정 수준에서 안정된 보기 상황은 항정상태로 부르는 것이 더 정확하다. 옳지 않다.
    3. 정적 피드백은 변화를 더 키우는 되먹임이다. 값이 일정하게 유지되는 보기와는 다르다. 옳지 않다.
    4. 부적 피드백은 변화를 되돌려 안정시키는 되먹임 기전이다. 결과로 나타난 안정 상태 자체를 가리키는 보기와는 층위가 다르다. 옳지 않다.
    🧮 풀이
    1. 甲은 21℃ 쾌적한 실내에서 최대 산소섭취량의 절반 강도로 60분을 달렸는데, 35분쯤 지나자 오르던 체온이 더 높아지지 않고 고원에 도달해 일정하게 유지되어, 열 생산과 열 발산이 균형을 이룬 상태를 몸소 경험하였다.
    2. 甲이 항상성을 떠올려 보면 체온과 혈당, 산도까지 몸속 환경 전체를 두루 일정하게 지키려는 큰 틀의 개념인데, 보기처럼 운동 중 한 값이 특정 수준에서 안정되는 대목을 콕 집을 때는 항정상태라 부르는 편이 더 들어맞았다.
    3. 甲이 정적 피드백의 예를 살펴보니 어떤 변화가 생기면 그것을 도리어 더 크게 밀어붙여 증폭시키는 되먹임이어서, 한 값이 오르내림 없이 일정하게 유지되는 보기의 상태와는 방향이 정반대로 어긋났다.
    4. 甲이 부적 피드백을 들여다보니 이것은 값이 흔들리면 원래대로 되돌려 안정시키는 조절 기전 자체를 가리키는데, 그렇게 조절한 끝에 나타난 안정된 결과 상태를 지목하는 보기와는 서로 다른 층위에 놓여 있었다.

    인체 내 포도당신생합성(gluconeogenesis)의 주요 기질(substrate)이 아닌 것은?

    1. 1. 염소(chlorine)
    2. 2. 젖산염(lactate)
    3. 3. 알라닌(alanine)
    4. 4. 글리세롤(glycerol)

    정답 1번

    🔍 보기별 해설
    1. 염소는 전해질 이온이지 포도당을 만드는 재료가 아니다. 포도당신생합성 기질이 아니다. 옳지 않다.
    2. 젖산염은 포도당신생합성의 대표적 기질이다. 무산소 대사로 생긴 젖산이 간에서 포도당으로 재합성되는 재료가 되므로 기질이 맞다. 옳지 않다.
    3. 알라닌은 아미노산으로 포도당신생합성 기질이다. 기질이 맞다. 옳지 않다.
    4. 글리세롤은 중성지방 분해 산물로 포도당신생합성 기질이다. 기질이 맞다. 옳지 않다.
    🧮 풀이
    1. 甲은 오래 굶어 간이 포도당을 새로 만들어 낼 때 젖산염·알라닌·글리세롤 같은 재료를 끌어다 쓴다는 것을 배웠는데, 전해질 이온인 염소는 이런 재료 목록에 들지 못해 포도당신생합성의 기질이 될 수 없음을 甲이 정리하였다.
    2. 甲이 숨이 가쁘도록 전력 질주를 한 뒤 근육에 쌓인 젖산이 간으로 실려 가는 길을 따라가 보니, 이 젖산염이 간에서 다시 포도당으로 짜여 올라가는 대표적인 재료로 쓰이고 있었다.
    3. 甲이 오래 굶어 근육 단백질이 조금씩 헐리는 흐름을 살펴보니, 거기서 떨어져 나온 아미노산인 알라닌이 간으로 보내져 포도당을 새로 짓는 재료로 요긴하게 쓰이고 있었다.
    4. 甲이 공복에 지방이 분해되는 과정을 지켜보니, 중성지방이 쪼개지며 나온 글리세롤이 혈액을 타고 간으로 흘러가 포도당을 새로 만드는 밑감으로 활용되고 있었다.

    그림은 골격근 섬유의 특징을 보여주고 있다. ㉠에 들어갈 변인으로 옳지 않은 것은?

    1. 1. ATPase 활성도
    2. 2. 최대 특이장력(specific tension)
    3. 3. 최대 수축 속도
    4. 4. 미토콘드리아 수

    정답 4번

    🔍 보기별 해설
    1. ATPase 활성도는 Type IIx가 Type I보다 높다. 그래프 모양(IIx가 큼)과 맞으므로 ㉠에 들어갈 수 있다. 옳지 않다.
    2. 최대 특이장력은 Type IIx가 더 크다. 그래프와 맞으므로 ㉠에 들어갈 수 있다. 옳지 않다.
    3. 최대 수축 속도는 Type IIx가 더 빠르다. 그래프와 맞으므로 ㉠에 들어갈 수 있다. 옳지 않다.
    4. 미토콘드리아 수는 Type I이 Type IIx보다 많다. IIx가 더 큰 그래프 모양과 반대이므로 ㉠에 들어갈 수 없다. 옳지 않다(정답).
    🧮 풀이
    1. 甲이 지근섬유와 속근섬유에서 ATP를 쪼개는 분해효소의 활성을 나란히 견주어 보니, 속근인 Type IIx 쪽이 지근인 Type I보다 그 활성이 훨씬 높게 나와, IIx 막대가 크게 솟은 ㉠ 그래프에 이 변인이 무리 없이 들어맞았다.
    2. 甲이 근섬유별로 단위 면적당 낼 수 있는 최대 힘인 특이장력을 재어 보니 속근 Type IIx가 지근 Type I보다 더 큰 값을 보여, IIx가 높게 그려진 ㉠ 자리에 이 변인이 자연스럽게 어울렸다.
    3. 甲이 두 근섬유가 얼마나 빠르게 오그라드는지 최대 수축 속도를 비교해 보니 속근 Type IIx가 지근 Type I보다 훨씬 빨라, IIx 막대가 우뚝한 ㉠ 그래프와 이 변인이 잘 들어맞았다.
    4. 甲이 지구력에 쓰는 지근섬유(Type I)와 순간적인 힘을 내는 속근섬유(Type IIx)를 현미경으로 견주어 보니, Type I 안에는 에너지 공장인 미토콘드리아가 훨씬 빽빽하게 차 있었고 Type IIx에는 그 수가 적어, IIx가 더 크게 그려진 ㉠ 자리에는 미토콘드리아 수가 도무지 어울리지 않았다.

    근육 세포에서 크레아틴 카이네이즈(Creatine Kinase, CK)의 활성도를 증가시키는 것은?

    1. 1. ATP 증가
    2. 2. NAD 감소
    3. 3. ADP 증가
    4. 4. 젖산 감소

    정답 3번

    🔍 보기별 해설
    1. ATP가 증가하면 굳이 ATP를 더 만들 필요가 없어 CK 활성은 오르지 않는다. 옳지 않다.
    2. NAD 감소는 CK 활성 증가의 직접 신호가 아니다. CK를 움직이는 방아쇠는 ADP의 증가이지 NAD 농도 변화가 아니므로 활성이 오르지 않는다. 옳지 않다.
    3. ADP가 늘면 ATP를 다시 만들 필요가 커져 CK 활성도가 증가한다. 옳다.
    4. 젖산 감소는 CK 활성 증가와 직접 관련이 없다. 크레아틴 카이네이즈의 활성을 끌어올리는 것은 ADP 증가이지 젖산 농도의 오르내림이 아니기 때문이다. 옳지 않다.
    🧮 풀이
    1. 甲이 근육 속 ATP가 넉넉히 차 있는 상태를 떠올려 보니, 이미 에너지가 충분해 굳이 더 만들 까닭이 없다 보니 크레아틴 카이네이즈도 서둘러 일할 이유가 없어 그 활성이 오르지 않았다.
    2. 甲이 근육 속 NAD가 줄어든 경우를 살펴보니, 이 변화는 크레아틴 카이네이즈를 곧바로 깨워 일하게 만드는 신호와는 거리가 멀어, NAD가 줄었다고 해서 이 효소의 활성이 따라 오르지는 않았다.
    3. 甲이 전력으로 질주하자 근육 속 ATP가 빠르게 쪼개져 찌꺼기인 ADP가 잔뜩 쌓였고, 늘어난 ADP를 신호로 삼은 크레아틴 카이네이즈가 크레아틴인산을 끌어다 ADP를 다시 ATP로 되돌리느라 활성도가 부쩍 높아졌다.
    4. 甲이 근육 속 젖산이 줄어드는 흐름을 짚어 보니, 이 변화는 크레아틴 카이네이즈가 더 바쁘게 일하도록 부추기는 것과는 직접 이어지지 않아, 젖산이 줄어도 이 효소의 활성은 그대로였다.

    4. 甲이 밤에 방의 불을 모두 끄고 눕자 솔방울샘에서 멜라토닌이 넉넉히 분비되어 스르르 잠이 들었지만, 자다 깨어 휴대폰 화면의 밝은 빛을 쬐자마자 멜라토닌 분비가 억제되면서 잠이 확 달아났다.
